## Ziel

`herold` macht Herold.at-Suchergebnisse im Terminal nutzbar. Suchbefehle geben immer stabile JSON-Ergebnisobjekte inklusive normalisierter Felder und `raw`-Daten aus der Herold-Seite aus.
